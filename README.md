# Java Log Analyzer

## Overview
This project analyzes application logs and identifies common errors such as:
- Exceptions
- Timeout issues
- General errors

## Features
- Reads log files
- Detects patterns
- Generates summary report

## Technologies Used
- Java
- File I/O
- Exception Handling

## How to Run
1. Compile:
   javac LogAnalyzer.java

2. Run:
   java LogAnalyzer

## Output
Displays count of errors, exceptions, and timeout issues.
